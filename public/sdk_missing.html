Camera SDK Not Installed

MagicQC requires the MagicCamera SDK to operate.
The measurement engine cannot start without it.

Installation Steps
1. Contact your system administrator
2. Install the MagicCamera SDK
3. Ensure the SDK is installed to the default path:
C:\Program Files (x86)\MindVision\
4. Restart this application after installation

The application will start normally once the SDK is detected.
